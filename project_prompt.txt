Project Details:
This will be simple youtube video layout rearranger plugin.  The goal is to make it more user friendly to watch videos and scroll comments on a narrow vertical window (think using 1/4 width for an ultrawide monitor).
The goal is for the video to be "full width" and pinned.  I want to be able to scroll through the comments below without the video moving.  I also want the recommended videos on the right side panel to be collapsable to the right.  So if I open it it's a popup style list.
